# Панель Жизни

Личный дашборд (PWA): планировщик с голосовым вводом, целями, идеями и заметками.
Тёмная тема с неоновыми подсветками, живая спираль на главной.

## Файлы проекта

- `index.html` — главная страница (карточки разделов + живая спираль)
- `planner.html` — экран Планировщика (задачи, статусы, календарь-месяц, детали)
- `styles.css` — общая дизайн-система
- `store.js` — хранилище данных (localStorage, общее для всех экранов)
- `manifest.json` + `service-worker.js` — для установки на телефон (PWA)
- `icon-192.png`, `icon-512.png` — иконки приложения

## Как выложить на GitHub Pages

1. Создай новый репозиторий на github.com (например, `panel-zhizni`). Можно публичный.
1. Загрузи в него ВСЕ файлы из этой папки (через сайт GitHub: Add file → Upload files, перетащи всё).
1. В репозитории: **Settings → Pages**.
1. В «Source» выбери ветку `main` и папку `/ (root)`, нажми Save.
1. Через минуту-две появится ссылка вида `https://ТВОЙ_ЛОГИН.github.io/panel-zhizni/`.

## Как поставить на айфон

1. Открой ссылку в **Safari** (именно Safari, не Chrome — установка на iOS работает только из него).
1. Кнопка «Поделиться» → **«На экран Домой»**.
1. На рабочем столе появится иконка со спиралью — открывается как приложение, без адресной строки.

## Важно про данные

Сейчас задачи хранятся в браузере устройства (localStorage). Это значит:

- данные не теряются между запусками на ОДНОМ устройстве;
- но НЕ синхронизируются между айфоном и компьютером.

Облачную синхронизацию (как в DesignFlow CRM через Supabase) можно добавить позже.

## Про голосовой ввод

Кнопка-микрофон в Планировщике распознаёт речь через Web Speech API.

- На компьютере в Chrome — работает на русском.
- На айфоне (Safari/Chrome на iOS) распознавание чаще недоступно — кнопка станет
  приглушённой с подсказкой. Это ожидаемо: на iOS используется системная диктовка
  клавиатуры (значок микрофона на клавиатуре) при обычном вводе текста.

## Обновление версии (про кэш)

Если после изменений на телефоне видна старая версия — подними версию кэша:
в `service-worker.js` поменяй `panel-zhizni-v1` на `-v2` и т.д.
(Знакомая по DesignFlow история с `designflow-v53`.)
